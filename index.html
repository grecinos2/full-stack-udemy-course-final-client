About This Site

This single-page website was created as final assignment on a course I am taking in regard to Full Stack Web development. It illustrates many concepts. Including, CSS, HTML with semantic tags, Javascript, Node.js with database integration, Bootstrap, Ajax, promises, modal popups, client server interaction, SQL, MySql, Xampp, PHP. The list goes on.

The assignment is to create a student management system. The requirements are to create a web frontend and backend based on Node.js. The frontend should demonstrate the use of HTML, CSS, and Javascript. The backend will use Node.js, Express, and MySQL. The purpose is to create a RESTful API working with a database and building a responsive front end. This website demonstrates all of the required features. I added pagination for good measure.
